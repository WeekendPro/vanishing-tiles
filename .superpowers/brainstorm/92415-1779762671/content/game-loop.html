Core Game Loop

Each round flows through 4 phases. The same loop repeats, getting harder each round.

PHASE 1
VIEWING
8×10 grid appears, mostly filled.
Empty slots are tetromino-shaped gaps.
Progress bar counts down (5s at lvl 1).
▼
grid hides
PHASE 2
SELECTING
Piece menu appears (all 7 tetrominoes).
User picks how many of each they need.
Progress bar counts down (15s at lvl 1).
Speed bonus accrues for unused time.
▼
timer ends
✓ selection fills grid exactly
PHASE 3A
AUTO-PLACE
Pieces snap into gaps
automatically. No penalty.
✗ selection doesn't fit
PHASE 3B
MANUAL PLACE
Grid reappears. Click piece → click cell. Rotation supported. Penalty applied.
▼
all pieces placed
PHASE 4
SCORING
Completed rows flash and clear.
Score breakdown shown briefly:
correctness + speed bonus + efficiency bonus + row clears
▼
Next round begins → harder timers, more/complex gaps
