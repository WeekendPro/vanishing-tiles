One tray, five slots — a set that rotates

Every phase presents the same shape of choice: a 5-item tray. Each level reveals an 8-tile sequence drawn from those 5 (repeats allowed). What changes between levels is the set itself — different shapes, a different emoji group, a different palette — so the run never settles into one fixed alphabet.
